DATING ALGORITHM ANALYSIS FOR TINDER

Project Details

Worked in groups of 5.

This project aimed to enhance the success rate of matches on the Tinder dating platform. Leveraging a dataset from experimental dating events spanning 2002-2004, we focused on analyzing key attributes influencing successful pairings. The dataset, originally comprising 122 features and 8378 rows, was streamlined to the 16 most relevant features to align with project goals. Preprocessing steps included handling missing values, consolidating age and race information, and applying feature engineering techniques. The resulting dataset was standardized to ensure consistency and relevance. Our proposed analysis involved employing various classification techniques, such as logistic regression, decision trees, Random Forest, and Gradient Boosting, to predict match outcomes during speed dating events.

The project aimed to improve the user experience on Tinder by refining matchmaking algorithms based on data-driven insights, ultimately leading to more meaningful connections for users.
